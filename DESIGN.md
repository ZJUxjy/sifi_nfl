# SIFI NFL - 科幻橄榄球经理游戏

## 项目概览
这是一个设定在未来殖民星球上的橄榄球模拟经营游戏。星球上最流行的运动是橄榄球（NFL规则），玩家扮演球队总经理的角色，管理球队的运营、战术、人员配置等。

## 世界背景故事

### 殖民星球历史
这颗殖民星球由三个殖民公司的开拓者创建：**Google宇航科技**、**地球理想公司**和**太阳系矿业公司**。最初星球被分割为三个国家，战争频繁。最后，一位自称耶稣转世之人创立的**救世军**统一了星球，建立君主立宪制度。皇帝必须由直系后代继承，无实际政治权利，仅作为国家象征。橄榄球作为星球上最流行运动也是因为皇家的运作。

### 星球概况
- **第一大陆**: 3亿人口，战争后由皇帝带领开发
- **第二大陆**: 5亿人口，同第一大陆
- **起源大陆**: 20亿人口，最繁华地区，首都**大都会**（6亿人口）坐落于此
- **矿业岛**: 8000万人口，矿产资源最丰富

大都会是人类有史以来最繁华的城市，是星球行政、经济、文化中心。

## 游戏世界设定

### 四大行政区

#### 1. 第一大陆 (First Continent)
- **人口**: 3亿
- **联盟类型**: 封闭联盟
- **球队数量**: 36支
- **联赛结构**:
  - 类似美国NFL
  - 封闭联盟，无升降级
  - 每年选秀（大学体系→职业体系）
  - 工资帽制度

#### 2. 第二大陆 (Second Continent)
- **人口**: 5亿
- **联盟类型**: 封闭联盟
- **球队数量**: 40支
- **联赛结构**: 类似第一大陆

#### 3. 起源大陆 (Origin Continent) - 大都会
- **人口**: 20亿
- **首都**: 大都会（6亿常驻人口）
- **联盟类型**: 开放联盟（升降级制度）
- **联赛结构**:
  - 3个并列顶级职业联赛（各12支球队）
  - 第一阶段: 每队联盟内打11场
  - 第二阶段（争冠组）: 每赛区前4名，打跨赛区8场
  - 保级组: 剩余24队，与其他赛区16队比赛
  - 季后赛: 双败淘汰制，中立场地
  - 降级规则: 每赛区最后一名直接降级

#### 4. 矿业岛 (Mining Island)
- **人口**: 8000万
- **资源**: 矿产最丰富
- **联盟类型**: 开放联盟（英超式升降级）
- **联赛结构**:
  - 超级联赛: 20队
  - 冠军联赛、A级联赛、B级联赛
  - 每年3个升降级名额
  - 转会制度: 转会费制度，类似现代足球

## 球员系统

### 球员属性 (Player Ratings)

#### 身体属性 (0-100)
- **身高 (HGT)**: 影响投掷视野、接球高度、防守范围
- **力量 (STR)**: 冲撞、阻挡、擒抱的力量
- **速度 (SPD)**: 冲刺、追击、回追速度
- **耐力 (ENDU)**: 持续作战能力、体能消耗恢复

#### 技术属性 (0-100)
- **视野 (THV)**: 四分卫阅读防守、预判能力
- **臂力 (THP)**: 传球距离和速度
- **精准度 (THA)**: 传球命中率
- **爆发力 (BSC)**: 加速度、突破能力
- **灵巧度 (ELU)**: 变向、假动作、逃脱擒抱
- **跑动 (RTR)**: 跑位选择、路线识别
- **接球 (HND)**: 接球稳定性、不丢球
- **传球阻挡 (PBK)**: 保护四分卫
- **跑动阻挡 (RBK)**: 开路、冲跑支援
- **抢球 (PCV)**: 抢断、破坏传球
- **擒抱 (TCK)**: 有效擒抱、扑球成功率
- **冲传 (PRS)**: 向四分卫施压
- **跑防 (RNS)**: 对抗跑动、对抗突破

#### 专项属性 (0-100)
- **踢球力量 (KPW)**: 射门、开球距离
- **踢球精准 (KAC)**: 射门、开球准确性
- **弃踢力量 (PPW)**: 弃踢距离
- **弃踢精准 (PAC)**: 弃踢准确性

#### 综合属性 (Composite Ratings)
- **传球评分**: (THV + THP + THA) / 3
- **跑动评分**: (BSC + ELU + RTR + SPD) / 4
- **接球评分**: (HND + SPD + ELU) / 3
- **阻挡评分**: (PBK + RBK + STR) / 3
- **冲传评分**: (PRS + STR + SPD) / 3
- **防跑评分**: (RNS + TCK + STR) / 3
- **防传评分**: (PCV + SPD + ELU) / 3
- **擒抱评分**: (TCK + STR + SPD) / 3
- **踢球评分**: (KPW + KAC) / 2
- **弃踢评分**: (PPW + PAC) / 2

#### 隐藏属性
- **潜力 (POT)**: 球员发展上限
- **伤病指数**: 伤病易感程度

### 球员类型 (Position Types)
- **QB**: 四分卫 (传球核心)
- **RB**: 跑卫 (地面进攻)
- **WR**: 外接手 (接球核心)
- **TE**: 近端锋 (混合位置)
- **OL**: 进攻锋线 (阻挡)
- **DL**: 防守锋线 (冲传)
- **LB**: 线卫 (混合防守)
- **CB**: 角卫 (防传)
- **S**: 安全卫 (最后防线)
- **K**: 踢球手 (射门)
- **P**: 弃踢手 (弃踢)
- **KR**: 开球回攻 (开球回攻)
- **PR**: 弃踢回攻 (弃踢回攻)

### 球员发展系统
- **年龄曲线**:
  - 18-23岁: 快速成长期，潜力大幅提升
  - 24-28岁: 成长期，稳步提升
  - 29-32岁: 巅峰期，保持巅峰
  - 33-36岁: 衰退期，缓慢下滑
  - 37岁以上: 快速衰退期

- **影响因素**:
  - 出场时间: 影响成长速度
  - 训练质量: 影响属性提升
  - 伤病: 影响长期发展
  - 精神属性: 影响成长潜力

## 合同与薪资系统

### 合同要素
- **年限**: 1-7年
- **薪资**: 每年金额（以千信用点为单位）
- **签约奖金**: 一次性支付
- **保障金**: 保障的金额部分
- **激励条款**: 基于表现的额外奖励

### 薪资制度

#### 封闭联盟（第一、二大陆）
- **工资帽**: 固定上限（200,000千信用点）
- **最低工资**: 必须达到底线（150,000千信用点）
- **奢侈税**: 超过上限2.5倍（250,000千信用点）触发
- **交易薪资匹配**: 交易需匹配125%薪资

#### 开放联盟（起源大陆、矿业岛）
- **无工资帽**: 可以自由签约球员
- **转会费**: 球员在球队间的交易费用
- **球员自由**: 合同期满即可自由转会

### 合同类型
- **新秀合同**: 选秀球员的首份合同
  - 起源大陆选秀球员: 新秀工资×2倍的中产合同
  - 原球队获得: 合同工资的10% + 竞拍费用

- **老将合同**: 自由球员市场签约
- **续约合同**: 与本队现有球员续签

## 财务系统

### 球队预算
- **收入来源**:
  - 票务收入: 主场门票销售
  - 转播收入: 联赛转播分成
  - 商品销售: 球队周边产品
  - 赞助收入: 商业赞助
  - 奖金: 联赛和杯赛奖金

- **支出项目**:
  - 球员工资: 主要支出
  - 教练组费用
  - 设施维护
  - 球场运营
  - 医疗费用

- **财政等级**:
  - 豪门: 无限制支出
  - 富裕: 高预算但有限制
  - 中等: 标准预算
  - 拮据: 预算受限
  - 破产: 财政危机

## 赛季管理系统

### 赛季阶段
1. **休赛期**:
   - 选秀
   - 自由球员市场
   - 训练营
   - 季前赛

2. **常规赛**:
   - 17场比赛
   - 伤病管理
   - 战术调整
   - 绩效评估

3. **季后赛**:
   - 不同赛区季后赛形式不同
   - 淘汰赛制
   - 中立场地或主客场

4. **休赛季**:
   - 球员休息
   - 训练提升
   - 合同到期处理

### 赛程生成
- 平衡主客场
- 考虑赛区对阵
- 轮换对手
- 特殊对阵（跨赛区、杯赛）

## 比赛引擎 (Match Engine)

### 基于zengm足球引擎改编
参考 `/home/xu/code/zengm/src/worker/core/GameSim.football/`

### 比赛机制
- **四节比赛**: 每节15分钟
- **加时赛**:
  - 常规赛: 双方各一次进攻机会
  - 季后赛: 15分钟加时，可多次
  - 突然死亡: 先得分者获胜

### 进攻类型
- **传球**: 基于QB传球评分 vs 防传评分
- **跑动**: 基于跑动评分 vs 防跑评分
- **射门**: 基于距离、踢球手评分
- **弃踢**: 基于弃踢手评分
- **开球**: 比赛开始或得分后
- **两分转换**: 达阵后的选择

### 比赛数据统计
- 球员数据: 传球码数、冲球码数、接球次数、擒抱、截球、达阵、射门
- 球队数据: 总码数、控球时间、转换成功率、得分、犯规

### 战术系统
- **进攻战术**:
  - 传球倾向: 平衡/传球主导/跑动主导
  - 进攻速度: 快/慢
  - 风险等级: 保守/激进

- **防守战术**:
  - 防守类型: 区域防守/人盯人
  - 冲传强度: 轻度/中度/重度
  - 逼抢程度: 被动/积极

## 选拔与选秀系统

### 大学选秀系统
- **选秀年龄**: 21-22岁
- **选秀轮数**: 7轮
- **选秀顺序**: 逆战绩顺序（最差球队先选）
- **选秀权交易**: 可以交易选秀权

### 起源大陆选秀（跨区球员转会）
- **参赛条件**:
  - 第一大陆: 球员年满25周岁
  - 第二大陆: 年满24周岁且职业体系内至少效力2年
  - 矿业岛: 职业体系下至少效力3年

- **选秀流程**:
  - 球员获得许可参与选秀
  - 顶级联赛球队竞价竞拍
  - 竞拍获胜者可签订新秀×2倍中产合同
  - 原球队补偿: 首份合同工资的10% + 竞拍费用
  - 联赛补偿: 按球员人数给其他地区职业协会补偿

### 球员生成
- **随机生成**: 基于正态分布
- **潜力分配**: 部分球员为高潜力
- **属性分布**: 根据位置类型分配权重
- **姓名生成**: 科幻风格的姓名系统

## 皇权杯 (Imperial Cup)

### 赛事规则
- **举办频率**: 每4年一次
- **参赛资格**:
  - 大都会地区: 当年前6名
  - 第一大陆: 前4名
  - 第二大陆: 前4名
  - 矿业岛: 前2名

- **赛制**: 淘汰赛制
- **场地**: 轮流在各赛区主办
- **商业价值**: 全星球最高商业价值

### 历史统计
- 100年间，25届皇权杯:
  - 大都会地区: 17次冠军
  - 矿业岛: 4次冠军
  - 第一大陆: 3次冠军
  - 第二大陆: 1次冠军
